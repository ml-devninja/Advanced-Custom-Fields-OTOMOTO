== Installation ==

1. Copy the `acf-field-otomoto` folder into your `wp-content/plugins` folder
2. Activate the OTOMOTO plugin via the plugins admin page
3. Create a new field via ACF and select the OTOMOTO type
4. Please refer to the description for more info regarding the field type settings
